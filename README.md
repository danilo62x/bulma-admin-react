# Bulma Admin / React

> O mesmo layout, agora em React 19

[![Licença: MIT](https://img.shields.io/badge/Licen%C3%A7a-MIT-blue.svg)](./LICENSE) ![Grátis](https://img.shields.io/badge/pre%C3%A7o-Gr%C3%A1tis-brightgreen)

Versão React do nosso admin Bulma. Mesma identidade visual, mesma estrutura — porém com hooks, Vite e TypeScript estrito. Ideal para quem prefere o ecossistema React.

## 🧱 Stack

- React 19
- TypeScript
- Vite
- Bulma
- Zustand

## ✨ Recursos

- Mesmas 12 views do template Vue
- React 19 com Hooks e Suspense
- React Router 7 e Zustand
- TypeScript estrito
- Vite + ESLint + Prettier
- Layout responsivo de mobile a 4K

## 🚀 Como rodar

Requer [Node.js](https://nodejs.org) 18+.

```bash
npm install
npm run dev
```

Para gerar a versão de produção: `npm run build` (saída em `dist/`).
